# DocNear - Doctor Booking System

A high-end, responsive Doctor Appointment Booking System featuring a modern "Liquid Glass" (Glassmorphism) aesthetic.

## Features

- **Search & Discovery**: Find doctors by name or specialty with a real-time search.
- **Specialist Categories**: Browse doctors by categories like Cardiology, Dermatology, etc.
- **Medical Branches**: Filter between Allopathy and Homeopathy.
- **Doctor Listings**: Detailed cards with availability, ratings, and fees.
- **Map Integration**: View doctor locations on an interactive map using Leaflet.js.
- **Booking System**: Book appointments with real-time slot validation.
- **Admin Dashboard**: Manage doctors and bookings (includes credentials).
- **Responsive Design**: Fully mobile-responsive interface.

## Tech Stack

- **Frontend**: HTML5, CSS3 (Glassmorphism), JavaScript (ES6+)
- **Map**: Leaflet.js (OpenStreetMap)
- **Icons**: FontAwesome 6

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in your browser.
   - OR run a local server for the best experience (e.g., `npx serve .`).
